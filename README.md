📌 Task Manager
🚀 Overview

A simple and modern task manager built with HTML, CSS, and JavaScript, using IndexedDB for local persistent storage.

✨ Features

## Add / Delete tasks
Mark tasks as Pending or Done
Live search (title + description)
Filter tasks (All / Pending / Done)
Data persists after refresh
Instant UI updates (Optimistic UI)
Keyboard shortcut (Ctrl + Enter)
🗄️ Storage
IndexedDB for persistent client-side storage
Async database operations
Reliable data handling without backend
⚙️ Core Concepts
CRUD operations
Event delegation
Async / Await
State management (in-memory + DB sync)
Optimistic UI updates
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
IndexedDB
