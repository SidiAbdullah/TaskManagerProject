🗝️ Task Manager Project
🚀 Overview

This is a fully client-side Task Manager application built using HTML, CSS, and JavaScript, powered by IndexedDB for persistent storage.

The application allows users to manage tasks efficiently with real-time updates, search, filtering, and a smooth modern user experience—without any backend or frameworks.

✨ Features
🧩 Core Functionality
Add new tasks
Mark tasks as Pending / Done
Delete individual tasks
Clear all tasks
Persist data even after page refresh
🗄️ Data Storage (IndexedDB)
Uses browser IndexedDB for long-term storage
Supports large structured datasets
Asynchronous and scalable storage system
🔄 CRUD Operations
Create tasks
Read tasks on load
Update task status
Delete tasks
⚡ Live UI Features
Instant UI updates (Optimistic UI)
Automatic rollback on failure
Smooth and responsive interactions
🔍 Search & Filtering
Live search by title and description
Filter tasks by:
All
Pending
Done
🎯 Performance Optimization
Event delegation for scalable event handling
In-memory state for fast rendering
Reduced DOM manipulation
🎨 User Experience Enhancements
Toast notifications for actions
Loading indicators during database operations
Confirmation before deleting
Color-coded task status:
🟡 Pending
🟢 Done
⌨️ Keyboard Shortcuts
Ctrl + Enter → Quickly add a task
🧠 Key Concepts Learned
IndexedDB database handling
Asynchronous JavaScript (async/await)
State management in frontend apps
Optimistic UI design
Event delegation pattern
Client-side architecture design
📦 Tech Stack
HTML5
CSS3
Vanilla JavaScript
IndexedDB API
🏁 Purpose

This project is designed to simulate real-world frontend architecture without frameworks, helping developers understand how modern applications work under the hood before moving to tools like React or Vue.
